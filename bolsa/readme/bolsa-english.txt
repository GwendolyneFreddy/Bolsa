Bolsa

Mod for Baldur's Gate II. Bolsa is a new merchant who (as the name says well) sells several containers, bags for objects that will not appear in your inventory, ammunition belts and a very useful Staff of the Magi, besides commun objects. (Decompress this file in your game folder and execute setup-bolsa.exe to install).
